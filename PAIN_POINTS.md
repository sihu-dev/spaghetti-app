# 실제 Pain Point 분석 - 중소 제조업체 카탈로그 제작

## 실제 시나리오

### 회사: (주)한국펌프
- 업종: 산업용 펌프 제조
- 규모: 직원 15명
- 제품: 워터펌프, 케미컬펌프 등 50종
- 문제: 해외 바이어 미팅 전 영문 카탈로그 급히 필요

---

## Pain Point #1: 제품 정보 입력이 너무 번거로움

### 현재 프로세스 (문제)
```
1. 사진 촬영 (또는 기존 사진 찾기)
2. 엑셀에 스펙 수기 입력
   - 모델명: KP-500A
   - 용량: 500L/min
   - 전압: 220V/380V
   - 재질: SUS304
   - 가격: 1,500,000원
3. 엑셀 → 디자이너에게 전달
4. 디자이너가 다시 InDesign에 수기 입력
5. 오타 발견 → 수정 → 재전달 (3일 소요)
```

### 실제 Pain
- **시간**: 제품 1개 입력에 10분 소요 (50개 = 8시간)
- **오류**: 엑셀 복붙 과정에서 오타 빈발
- **중복 작업**: 영업팀 입력 → 디자이너 재입력

### 해결 방안: AI 자동 추출
```javascript
입력: 제품 사진 (명판/스티커 포함)
     ↓
AI Vision (Claude 3.5 Sonnet)
     ↓
출력: {
  "modelName": "KP-500A",
  "capacity": "500L/min",
  "voltage": ["220V", "380V"],
  "material": "SUS304",
  "features": ["내부식성", "고효율"]
}
```

**구현 우선순위: P0 (최우선)**

---

## Pain Point #2: 사진 품질이 제각각

### 현재 프로세스 (문제)
```
1. 공장에서 스마트폰으로 촬영
   - 배경: 창고 바닥, 지저분함
   - 조명: 어둡거나 과다 노출
   - 각도: 제각각

2. 디자이너 요청사항
   - "배경 제거해주세요"
   - "밝게 보정해주세요"
   - 추가 비용 발생
```

### 실제 Pain
- **비용**: 배경 제거 외주 (사진 1장당 5,000원)
- **시간**: 보정 요청 → 재촬영 → 2~3일 지연
- **불만족**: 그래도 퀄리티 낮음

### 해결 방안: AI 자동 보정
```javascript
입력: 날것 제품 사진
     ↓
1. 배경 제거 (remove.bg API 또는 SAM2)
2. 밝기/대비 자동 조정
3. 그림자 추가 (전문적 느낌)
     ↓
출력: 깔끔한 제품 사진
```

**구현 우선순위: P1**

---

## Pain Point #3: 브랜드 통일성 없음

### 현재 프로세스 (문제)
```
1. 영업팀: PPT로 간단히 제작
   - 빨강, 파랑, 초록 막 사용
   - 폰트도 제각각 (굴림, 돋움, 맑은고딕)

2. 디자이너 외주
   - "어떤 색 쓸까요?" → "음... 파란색이요?"
   - 매번 다른 톤의 파란색 사용

3. 결과: 회사마다 아니라 카탈로그마다 다른 느낌
```

### 실제 Pain
- **브랜드 인식 저하**: 바이어가 회사 기억 못함
- **비전문적**: "동네 인쇄소에서 만든 것 같다"
- **재사용 불가**: 명함, 웹사이트와 색상 불일치

### 해결 방안: 로고에서 브랜드 컬러 추출 (기존 코드 활용)
```javascript
입력: 회사 로고 이미지
     ↓
HCT 컬러 추출 (extraction.ts 활용)
     ↓
출력: {
  primary: "#2E5C8A",
  colorScale: { 50: "#E8F0F7", ..., 900: "#0A1F35" },
  typography: "Pretendard",
  brandStyle: "professional" // AI 판단
}
     ↓
모든 카탈로그에 자동 적용
```

**구현 우선순위: P0 (기존 코드 있음)**

## Pain Point #4: 다국어 번역 비용 폭탄

### 현재 프로세스 (문제)
```
한국어 카탈로그 완성 (100만원)
     ↓
영어 번역 필요 (바이어 미팅)
     ↓
번역 업체 견적
- 번역: 50만원
- 디자인 재작업: 30만원
     ↓
중국어도 필요? → +80만원
```

### 실제 Pain
- **비용**: 언어 1개당 80만원 추가
- **시간**: 번역 1주 + 디자인 1주 = 2주
- **긴급 대응 불가**: "내일 바이어 온다는데..."

### 해결 방안: AI 전문 번역 + 자동 레이아웃
```javascript
입력: 한국어 카탈로그 데이터
     ↓
Claude API (전문 용어 번역)
{
  ko: "스테인리스 스틸 케미컬 펌프",
  en: "Stainless Steel Chemical Pump",
  zh: "不锈钢化工泵",
  ja: "ステンレス鋼ケミカルポンプ"
}
     ↓
자동 레이아웃 조정 (영어는 길어서 폰트 크기 조정)
     ↓
PDF 자동 생성 (언어별)
```

**구현 우선순위: P1**

## Pain Point #5: 급하게 수정해야 하는데 디자이너 연락 안됨

### 현재 프로세스 (문제)
```
상황: 내일 전시회, 제품 가격 변경됨
     ↓
1. 디자이너 연락 (저녁 9시)
   - 전화 안받음

2. 다른 디자이너 찾기
   - 원본 파일(AI, PSD) 없음
   - "처음부터 다시 만들어야 해요" (50만원)

3. 포기하고 기존 카탈로그 사용
   - 바이어에게 "가격은 따로 말씀드릴게요" (비전문적)
```

### 실제 Pain
- **Lock-in**: 특정 디자이너에게 종속
- **원본 파일**: 받아도 Illustrator 못 다룸
- **긴급 수정 불가**: 주말/야간 대응 불가

### 해결 방안: 셀프 수정 가능한 웹 에디터
```javascript
카탈로그 생성 후 → 웹 에디터에서 언제든 수정
- 가격 변경: 클릭 → 입력 → 즉시 반영
- 제품 추가: 사진 업로드 → 자동 배치
- PDF 재생성: 1분
     ↓
디자이너 없이 즉시 대응
```

**구현 우선순위: P2**

---

## Pain Point #6: 인쇄소 납품용 파일 만들기 복잡함

### 현재 프로세스 (문제)
```
1. 디자이너가 준 PDF
2. 인쇄소: "300dpi CMYK로 주세요"
3. 영업팀: "???" (이해 못함)
4. 다시 디자이너에게 요청 (+5만원)
```

### 해결 방안: 프리셋 내보내기
```javascript
내보내기 옵션:
[ ] 웹용 (PDF, 72dpi, RGB)
[ ] 인쇄용 (PDF, 300dpi, CMYK, 3mm 여백)
[ ] 이메일용 (저용량 PDF, 2MB 이하)
[ ] SNS용 (PNG, 1080x1080)
```

**구현 우선순위: P2**

## MVP 기능 우선순위 정리

### P0 (코어 기능 - 없으면 제품 아님)
1. ✅ **제품 사진 업로드**
2. ✅ **AI 제품 정보 추출** (모델명, 스펙)
3. ✅ **로고 → 브랜드 컬러 추출** (기존 코드)
4. ✅ **자동 카탈로그 생성** (1개 템플릿)
5. ✅ **PDF 다운로드**

### P1 (차별화 - 이것 때문에 고객이 선택)
6. ✅ **AI 영어 번역**
7. ⏳ **배경 제거/보정**
8. ⏳ **산업별 템플릿** (3종)

### P2 (Nice to have)
9. ⏳ **웹 에디터 (수정 기능)**
10. ⏳ **중국어/일본어 번역**
11. ⏳ **인쇄용 내보내기**

## 실제 사용 시나리오 (MVP)

```
1. 회원가입 (회사명, 이메일)
   ↓
2. "카탈로그 만들기" 클릭
   ↓
3. 로고 업로드 → AI가 브랜드 컬러 추출 (5초)
   ↓
4. 제품 사진 10장 업로드 (드래그)
   ↓
5. AI가 각 사진에서 정보 추출 (30초)
   - 사용자는 잘못된 부분만 수정
   ↓
6. "영어로도 만들기" 체크
   ↓
7. "생성하기" 버튼 클릭 (1분 대기)
   ↓
8. 미리보기 확인
   ↓
9. PDF 다운로드 (한국어.pdf, English.pdf)
   ↓
완료! (총 소요 시간: 5분)
```

**기존 방식: 2주 + 100만원**
**Catalog AI: 5분 + 월 5만원**
