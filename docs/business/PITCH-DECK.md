## 1. 문제 정의

**핵심 페인포인트:**
- 디자인 토큰 수작업 생성: **평균 40시간/프로젝트**
- 디자이너-개발자 소통 비용: **프로젝트의 30%**
- 멀티 플랫폼 변환: iOS/Android/Web 각각 수작업

**시장 검증:**
- 90%의 디자인 시스템 실패 (채택 부족)
- 70%의 개발자가 AI 코드 수정 필요

## 2. 솔루션

**SPAGHETTI = 이미지 → 디자인 토큰 자동화**

```
📸 이미지 업로드 → 🤖 AI 분석 → 📦 토큰 생성 → ⚡ 40시간 → 1분
```

1. Chrome Extension (설치 5초)
