# 🍝 SPAGHETTI

> **S**tructured **P**latform for **A**utomated **G**enerative **H**igh-quality **E**ngineered **T**emplate **T**ool **I**nterface
>
> AI 기반 디자인 시스템 자동화 도구. v0.dev의 한계를 넘어, 구조(Structure)를 인식하고 디자인 시스템 부품으로 완벽하게 조립합니다.
>
> ## 📁 프로젝트 구조
>
> ```
> spaghetti-app/
> ├── frontend/                 # React 프론트엔드
> │   ├── src/
> │   │   ├── components/       # 재사용 컴포넌트
> │   │   ├── pages/            # 페이지 컴포넌트
> │   │   ├── styles/           # CSS 스타일
> │   │   ├── App.jsx
> │   │   └── main.jsx
> │   ├── package.json
> │   └── vite.config.js
> │
> ├── backend/                  # Node.js 백엔드
> │   ├── src/
> │   │   ├── controllers/      # 컨트롤러
> │   │   ├── routes/           # API 라우트
> │   │   ├── services/         # 비즈니스 로직
> │   │   ├── models/           # 데이터 모델
> │   │   └── index.js
> │   ├── package.json
> │   └── .env.example
> │
> └── README.md
> ```
>
> ## 🚀 시작하기
>
> ### Frontend
>
> ```bash
> cd frontend
> npm install
> npm run dev
> ```
>
> ### Backend
>
> ```bash
> cd backend
> npm install
> npm run dev
> ```
